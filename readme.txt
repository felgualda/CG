Instruções para executar o trabalho:

- Abra a pasta CG-2D pela sua IDE de preferência
- Execute o arquivo run.py
- Realize as ações disponibilizadas no terminal e veja suas transformações acontecendo em tempo real na janela aberta

Obs.: caso queira editar o grupo de pontos que geram o vetor, use o arquivo gato.txt, que corresponde ao objeto do gato escolhido para este trabalho. Cada grupo com grau de
liberdade diferente está representado por um /nome_do_grupo seguido das coordenadas de seus pontos.
